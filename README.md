# seat-booking-app

座席予約 WebApp

Frontend:React  
Backend: Node.js  
API&Route:Express  
DataBase:MongoDB

##Backend Install

```bash
cd backend
npm install
npx nodemon app.js
```

##Frontend Install

```bash
cd frontend
npm install
npm start
```
